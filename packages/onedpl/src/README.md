## About oneDPL

oneDPL (oneAPI DPC++ Library) provides high-productivity APIs for parallel
programming with SYCL. It includes parallel STL algorithms with SYCL execution
policies for GPU acceleration.

For more information, see the [oneDPL documentation](https://uxlfoundation.github.io/oneDPL).
